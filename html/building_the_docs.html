Building the documentationÂ¶

Before you can build the documentation, you must install Sphinx. See the installation instructions on the Sphinx website for detailed information.

To build the documentation, complete the following steps:

  1. Open a command window in the doc folder.

  2. Enter make html to build the HTML output.

    Tip

    If you get an error message that the make command cannot be found, you might be in the wrong folder, or you might need to explicitly state which command to run (for example, ./make.bat html on Windows).

    On Linux, you might need to install make as instructed.

The documentation build reports any errors or warnings that occur. The output is placed in the doc/_build/html folder. Open doc/_build/html/index.html to start browsing the documentation.
