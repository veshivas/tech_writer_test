.. ota_dfu:

Over-the-air device firmware update (OTA-DFU)
#############################################

The device provides you with several options for upgrading its firmware.

To perform a device firmware update (DFU) with custom firmware, you must first generate a DFU package.
See :ref:`preparing`.

Then, update the firmware by following the instructions in :ref:`updating_firmware`.
